# 🌐 context-graph-compressor - Turn long chats into portable graphs

## 📌 What this app does

This tool helps you manage long conversations with artificial intelligence. When you talk to an AI for a long time, it starts to lose track of details. This software reads your conversation logs and maps them into a logical graph. 

You can save this graph and move it between different AI programs. This keeps your context intact. You spend less time explaining your history to the chat tool and more time solving problems.

## 💻 System requirements

You need a computer running Windows 10 or Windows 11. The app works on 64-bit systems. Make sure you have at least 200 megabytes of free space on your hard drive. You do not need to install other coding tools or complex software to run this app.

## 🛠 Using the tool

The main interface shows a simple dashboard. Follow these steps to compress your data:

1. Locate your chat history file. Most chat tools allow you to export your data as a JSON file.
2. Drag your JSON file into the main window of the app.
3. Click the button labeled Generate Graph.
4. Wait for the progress bar to finish.
5. Retrieve your finished graph from the output folder.

The app uses smart logic to find the core points of your chat. It removes repetitive text and organizes your history by topic. This makes the data small enough for any AI to read without confusion.

## 🔍 Understanding the process

The software uses a specific method to organize data. Instead of keeping a long list of messages, it builds a web of nodes. Each node represents a specific topic or decision from your conversation. 

If your conversation includes many turns, the tool acts as a filter. It identifies which parts of the chat matter most for your goals. The exported JSON file contains these weighted nodes. You can upload this file into other AI agents to jump straight back into your work.

## 📦 Handling errors

If the app fails to process a file, check the format of your chat history. The app requires a standard JSON structure. If your chat program offers different export formats like CSV or TXT, those will not work directly. Convert your files to JSON using basic text tools before you run them through the compressor.

Ensure your internet connection remains stable when the app verifies updates. You may see a small icon in the top right corner if a newer version exists. 

## 🛡 Security and privacy

The app runs locally on your computer. Your conversation data never leaves your machine. The code does not send your personal chats to external servers. You keep full control over your information at all times. 

## ❓ Frequently asked questions

Do I need to be a developer?
No. You do not need to read code or open a terminal. The interface relies on clicks and simple file movement.

Can I use this for multiple AI services? 
Yes. The output format works with major AI models. As long as the AI can accept a conversation state file, this graph will function as intended.

Does the file size affect the speed? 
Large files take longer to compress. A chat with thousands of messages might take a few seconds to process. The app shows a progress indicator so you know the status of your task. 

Where does the app save my graphs? 
The app creates a folder named Graphs inside your Documents folder. You can change this path in the settings menu.

Does this work on Apple computers? 
This version is for Windows only. The current download file requires the Windows operating system to function.

## 💡 Troubleshooting tips

- Restart the app if it stops responding during a large file import.
- Check for Windows updates if you see errors regarding missing system files.
- Ensure your antivirus does not block the application from creating the output folder.
